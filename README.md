# Pink-Serpent
Задачка с визуализацией на Cocos2d-x

сборка
```
git clone https://github.com/Ingener74/Pink-Serpent.git
cd Pink-Serpent
git submodule update --init
cd cocos2d-x
git checkout cocos2d-x-3.10
cd ../
mkdir build
cd build
cmake .. -DCMAKE_BUILD_TYPE={Debug|Release}
make -j4
./PinkSerpent
```
